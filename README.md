## 项目说明

1. 一般活动放入 `/src/activitys/` 下,活动目前以时间命名
1. 实时浏览器刷新,请启动 `$ gulp server`.默认访问 `localhost:3000/`
1. `$ gulp default` task 显示了相关帮助
